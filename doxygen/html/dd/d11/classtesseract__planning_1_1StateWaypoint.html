◆ position

Eigen::VectorXd tesseract_planning::StateWaypoint::position

The joint position at the waypoint.

This is different from waypoint because it can be cartesian or joint and this stores the joint position solved for the planned waypoint. This also can be used for determining the robot configuration if provided a cartesian waypoint for generating a native robot program.
